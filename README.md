# project_quintet

## How to run backend

`cd backend`

### Prequisities

### Mongo

Install mongo globally https://docs.mongodb.com/manual/administration/install-community/

Or run in docker

```
mkdir ~/data
sudo docker run -d -p 27017:27017 -v ~/data:/data/db mongo
```

### Python

Install python

```
pip install numpy
```

### Node

Intall node

```
npm i -g nodemon

```

## Run

`nodemon index.js`
or just
`node index.js`
